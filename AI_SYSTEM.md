# 🤖 Система ИИ — Полноценный противник на карте

## ✅ Реализовано

### 🏗️ Экономическая фаза (каждый ход)

| Действие | Описание |
|----------|----------|
| **Передача из гарнизона** | Всё из гарнизона передаётся герою |
| **Строительство зданий** | До 2 зданий за ход по приоритетам |
| **Найм существ** | Всё что доступно и можем оплатить |
| **Передача армии** | Новая армия снова передаётся герою |

### 🗺️ Фаза действий на карте

| Цель | Приоритет | Условие |
|------|-----------|---------|
| 🎯 Атака героя игрока | 150 | Если армия ИИ на 10% сильнее |
| 🏰 Захват города игрока | 140 | Если город в 25 клетках |
| 🛡️ Защита своего города | 120 | Если армия слабая |
| ⛏️ Захват шахт | 90 | Любые шахты рядом |
| ⚔️ Атака нейтральных существ | 70 | Если армия сильнее |
| ✨ Сбор артефактов | 75 | Видит в 12 клетках |
| 💰 Сбор ресурсов | 60 | Видит в 10 клетках |
| 🔍 Исследование | 40 | Если других целей нет |
| 🏠 Возврат в город | 20 | Слабая армия далеко от дома |

### 📊 Система приоритетов

```
ATTACK_WEAK_HERO:  150  ← Атаковать слабого героя игрока
CAPTURE_ENEMY_TOWN: 140  ← Захватить город игрока
DEFEND_OWN_TOWN:   120  ← Вернуться защитить свой город
CAPTURE_MINE:       90  ← Захватить шахту
HIRE_STRONG:        85  ← Нанять сильных существ
COLLECT_ARTIFACT:   75  ← Подобрать артефакт
ATTACK_CREATURE:    70  ← Атаковать нейтралов
COLLECT_RESOURCE:   60  ← Подобрать ресурс
BUILD_NORMAL:       50  ← Построить здание
EXPLORE:            40  ← Исследовать
RETURN_TO_TOWN:     20  ← Вернуться в город
```

## 🏗️ Приоритет строительства

ИИ строит здания в таком порядке:

1. **Цитадель** — базовое здание
2. **Казармы** — ополченцы (Haven)
3. **Стрельбище** — лучники
4. **Кузница** — покупка артефактов
5. **Рынок** — обмен ресурсов
6. **Башня грифонов** — грифоны
7. **Собор** — ангелы (требует много ресурсов)
8. **Гильдия магов** — заклинания
9. **Стены** — защита
10. **Проклятый храм** — скелеты (Necropolis)
11. **Склеп** — зомби
12. **Мавзолей** — вампиры

ИИ **проверяет**:
- ✅ Фракция здания совпадает с фракцией ИИ (или common)
- ✅ Все требования выполнены (предыдущие здания построены)
- ✅ Хватает ресурсов

## ⚔️ Логика боя ИИ с игроком

### Когда ИИ атакует героя:

```typescript
if (aiPower > playerPower * 1.1) {
  // Сильнее на 10%+ → атакует!
  priority = ATTACK_WEAK_HERO (150);
}
else if (aiPower > playerPower * 0.7 && distance <= 8) {
  // 70% силы, но близко → рискнём
  priority = ATTACK_STRONG_HERO (30);
}
```

### Что происходит при атаке:
1. 🚨 Появляется уведомление: *"⚔️ Лорд Мортис нападает на вас!"*
2. 🛑 Движение игрока останавливается
3. ⚔️ Запускается бой (ИИ — атакующий, игрок — защитник)
4. 🏆 Победитель получает опыт и ресурсы

### Что делает ИИ с нейтралами:
- Если сила ИИ > 100 → автоматически побеждает
- Получает 200 опыта за каждого
- Проверяет повышение уровня

## 🎮 Советы игроку

### 🔴 Как защититься от ИИ:
1. **Стройте стены** — ИИ не сможет легко захватить город
2. **Оставляйте гарнизон** — даже слабые существа защитят город
3. **Следите за уведомлением** — *"Лорд Мортис захватил шахту"*
4. **Атакуйте первыми** — если ИИ слабее, атакуйте его героя
5. **Не оставляйте героя далеко от города**

### 🟢 Как использовать ИИ:
1. **Захватывайте его шахты** — они дают доход ИИ
2. **Атакуйте его героя** — получите опыт и ресурсы
3. **Исследуйте в его сторону** — находите его города
4. **Захватывайте его города** — условие победы

## 🚀 Как работает интеграция с WorldScene

### Инициализация:
```typescript
this.aiSystem = new AISystem(map, objects, {
  getVictorySystem: () => this.victorySystem,
  getPlayerHero: () => this.hero,
  getPlayerPosition: () => this.getHeroPosition(),
  getTownData: (id) => this.getTownDataForAI(id),
  onAIAttackHero: (h) => this.handleAIAttackHero(h),
  onAIMove: (id, x, y) => this.animateAIHeroMove(id, x, y),
  // ... и другие коллбэки
});
```

### Каждый ход:
```typescript
public endTurn(): void {
  // ... логика игрока ...
  
  // Ход ИИ
  this.aiSystem.executeTurn();
  this.updateAIHeroPositions();
  
  // Проверка условий победы
  this.checkVictoryConditions();
}
```
