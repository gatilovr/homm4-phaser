### 📊 Расчёт опыта (НОВОЕ!)

**Формула:**
```
Опыт = Σ(боевая_мощь_убитых × 10)
Минимум: 100 опыта за бой
```

Боевая мощь:
```
(средний_урон × количество + HP × количество) / 2
```

Пример:
- 15 гоблинов (урон 1-2, HP 5) = 15 × (1.5 + 5) / 2 × 10 = 487 опыта
- 3 огра (урон 8-14, HP 40) = 3 × (11 + 40) / 2 × 10 = 765 опыта

## 🤖 ИИ в бою

### Стратегии ИИ:
1. **Стрелок** — стреляет с расстояния, отступает если подошли
2. **Летающий** — сразу летит к слабому врагу
3. **Кавалерия** — бежит издалека для бонуса разбега
4. **Ближний бой** — идёт к приоритетным целям

### Приоритеты целей:
1. Вражеские стрелки (высокая угроза)
2. Вражеские герои (главная цель)
3. Слабые юниты (можно добить)
4. Ближайшие (если ничего лучше нет)

## 📈 Формулы урона

### Базовый расчёт:
```
baseDamage = random(min, max) × count × modifier_atk_def
modifier = max(0.3, 1 + (ATK - DEF) × 0.05)
```

### Модификаторы:
- **Благословение**: ×1.2+ урона
- **Жажда крови**: +5 атаки
- **Щит**: ×0.7 получаемого урона
- **Каменная кожа**: +5 защиты
- **Кавалерия**: +5% за клетку расстояния
- **Стрелок вблизи**: ×0.5 урона
- **Крит**: ×2 урона
- **Промах**: ×0.5 урона

## 🚀 Как использовать

### Обычный бой:
1. Найдите нейтральное существо на карте
2. Кликните — герой подойдёт и начнётся бой
3. Кликайте по зелёным клеткам для движения
4. Кликайте по красным для атаки
5. Используйте магию (M), защиту (D), ждать (W)

### Осада города:
1. Подойдите к вражескому городу
2. Бой начнётся автоматически как "Осада"
3. Сначала разрушьте стены/башни
4. Затем атакуйте защитников
5. После победы — город ваш!

### Некромантия:
1. Играйте за Некроманта
2. Выигрывайте бои
3. После боя увидите: "☠️ Некромантия: восстали 5 скелетов, 2 зомби"
4. Новая нежить добавится в армию
