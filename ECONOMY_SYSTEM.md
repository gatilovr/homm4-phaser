## 🎯 7 новых механик

### 1. ⛏️ **Шахты 7 типов ресурсов** (критическое исправление!)

**Было:** Все шахты давали +500 золота (ошибка)  
**Стало:** Каждая шахта даёт свой тип ресурса

| Тип шахты | Иконка | Ежедневный доход |
|-----------|--------|------------------|
| Золотая шахта | 💰 | 1000 золота |
| Лесопилка | 🪵 | 2 дерева |
| Рудник | ⛏️ | 2 руды |
| Хрустальная шахта | 💎 | 1 кристалл |
| Самоцветная шахта | 💠 | 1 самоцвет |
| Серная шахта | 🟡 | 1 сера |
| Ртутная шахта | 🩸 | 1 ртуть |

**Распределение:** золото 35%, дерево 18%, руда 18%, редкие 29%

### 2. 📈 **Еженедельный прирост существ** (критическое!)

**Было:** Существа для найма не пополнялись  
**Стало:** Каждую неделю — новые существа в жилищах

```
Неделя 1: ⏭️ Конец хода (7 раз)
📅 Новая неделя!
📈 Прирост в городе:
  • 14 Ополченцев
  • 9 Лучников
  • 7 Грифонов
```

**Бонусы:**
- 🏰 Цитадель: **+25%** к приросту
- ⬆️ Улучшенное жилище: **+50%** к приросту

### 3. ⬆️ **Апгрейды существ в городе** (новое!)

Теперь можно улучшать существ в армии за ресурсы:

| От кого → В кого | Стоимость |
|------------------|-----------|
| Пикинёр → Алебардщик | 💰1000 + 🪵2 |
| Лучник → Арбалетчик | 💰1500 + 🪵3 |
| Ангел → Архангел | 💰8000 + 💎5 + 🩸3 |
| Скелет → Скелет-воин | 💰800 + ⛏️2 |
| Костяной дракон → Призрачный дракон | 💰10000 + 🟡5 + 🩸3 |
| ... и ещё 30+ апгрейдов! |

### 4. 💱 **Улучшенный рынок** (обмен ресурс-ресурс)

**Было:** Только продажа за золото  
**Стало:**
- 💰 **Продажа:** любой ресурс → золото
- 🛒 **Покупка:** золото → любой ресурс
- 🔄 **Обмен:** ресурс ↔ ресурс (через золото)

**Курсы:**
| Ресурс | Покупка | Продажа |
|--------|---------|---------|
| Дерево/Руда | 750 | 400 |
| Кристаллы/Самоцветы/Сера/Ртуть | 1500 | 900 |

### 5. 🧙 **Магическая гильдия** (новое!)

В городе можно покупать заклинания (требует Mage Guild):
- 📜 Уровень 1 → 500 золота
- 📜 Уровень 2 → 1000 золота
- 📜 Уровень 3 → 1500 золота

**3-5 случайных заклинаний** обновляются каждую неделю!

### 6. 💵 **Ежедневный доход города** (новое!)

Город сам по себе даёт золото:
- 🏰 **Базовый доход:** 250 золота/день
- 🏛️ **Казначейство:** +500 золота/день
- 🏛️ **Капитолий:** +2000 золота/день
- 🏪 **Рынок:** +250 золота/день
- 🍺 **Таверна:** +50 золота/день

**Итого:** город без построек = 250 золота, с капитолием = 2750 золота!

### 7. 🚚 **Караваны между городами** (новое!)

- Передача существ между городами
- Время доставки зависит от расстояния (1 день на 5 клеток)
- Прогресс каравана виден в UI
- Можно отменить караван

## 🎮 Как проверить

### Запустите:
```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
npm run dev
```

### В игре:

1. **⛏️ Шахты** — захватите шахту, увидите тип (дерево/руда/кристаллы)
2. **📈 Прирост** — нажмите "Конец хода" 7 раз → появится "Новая неделя!" с приростом
3. **⬆️ Апгрейды** — наймите пикинёров → в городе появится вкладка "Апгрейды"
4. **💱 Рынок** — постройте Рынок → обменивайте ресурсы друг на друга
5. **🧙 Гильдия** — постройте Mage Guild → покупайте заклинания
6. **💵 Доход** — смотрите уведомления в начале дня: "Город принёс 1500 золота"

## 💡 Советы по игре

### Стартовая стратегия:
1. День 1: Постройте **Казармы** → наймите пикинёров
2. День 2: Захватите ближайшую шахту (любого типа!)
3. День 3-7: Собирайте ресурсы на карте
4. День 7 (неделя 2): Получите прирост → наймите ещё
5. День 8: Постройте **Рынок** → продайте лишнее дерево/руду
6. День 10: Постройте **Магическую гильдию** → купите заклинания
7. День 14: Улучшите пикинёров → алебардщики
8. День 20+: Атакуйте вражеский город!

### Что добывать в первую очередь:
- 💰 **Золото** — нужно для всего
- 🪵 **Дерево** — нужно для Рынка, стрельбища
- ⛏️ **Руда** — нужно для Кузницы, Цитадели
- 💎 **Кристаллы** — нужны для апгрейдов высших тиров
