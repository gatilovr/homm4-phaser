## 🎯 ContentManager — единый API

Создан **централизованный менеджер контента** с методами:

```typescript
// Существ
contentManager.getCreature('angel')
contentManager.getCreaturesByFaction('haven')        // 14 существ
contentManager.getCreaturesByTier(7)                  // Ангелы, драконы...
contentManager.getUpgradeCreature('pikeman')          // Алебардщик
contentManager.isCreatureAvailable(faction, id, built) // Проверка найма

// Зданий
contentManager.getBuilding('haven_barracks')
contentManager.getAvailableBuildings('haven', built)  // Что можно построить
contentManager.canAffordBuilding(resources, building)

// Фракций
contentManager.getFaction('haven')
contentManager.getAllFactions()                        // 6 фракций
contentManager.getFactionCreatures('haven')
contentManager.getFactionBuildings('haven')

// Заклинаний
contentManager.getSpell('fireball')
contentManager.getSpellsBySchool('fire')
contentManager.getSpellsByMageGuildLevel(3)

// Артефактов
contentManager.getArtifact('sword_of_might')
contentManager.getArtifactsByRarity('relic')
contentManager.getRandomArtifact('major')
```
