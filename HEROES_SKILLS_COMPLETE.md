## 🔧 Интеграция в другие модули

### ⚔️ BattleScene — навыки в бою

```typescript
// Урон ближнего боя с учётом Наступления + специализации barbarian
baseDamage *= heroManager.getMeleeDamageMultiplier(attackerHero);

// Урон стрелков с учётом Стрельбы + специализации ranger
baseDamage *= heroManager.getRangedDamageMultiplier(attackerHero);

// Снижение получаемого урона (Оборона)
baseDamage *= heroManager.getIncomingDamageMultiplier(defenderHero);

// Урон заклинаний (Колдовство + heretic)
const effectiveSpellPower = spellPower * heroManager.getSpellDamageMultiplier(hero);

// Регенерация маны за ход (Мистицизм)
hero.mana += heroManager.getManaRegenPerTurn(hero);

// UI выбора навыка при повышении уровня
heroManager.processLevelUp(scene, hero, notification);
```

```typescript
// Создание героя с правильными навыками и специализацией
```

## 🎮 Как проверить

### Запустите dev-сервер:
```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
npm run dev
```

### Тестируйте навыки:

1. **📋 Посмотреть навыки героя** — нажмите **H**
   - Откроется полная панель с характеристиками
   - Список навыков с уровнями
   - Активные бонусы

2. **⚔️ Проверить боевые навыки**
   - Атакуйте нейтрала
   - Герой с Наступлением наносит больше урона
   - Герой с Оборона получает меньше урона
   - Стрелки с Стрельба наносят +10-50% урона

3. **🧙 Проверить магические навыки**
   - Используйте заклинания
   - Колдовство увеличивает урон заклинаний
   - Мистицизм восстанавливает ману каждый ход
   - Интеллект даёт больше максимальной маны

4. **🗺️ Проверить приключенческие навыки**
   - Разведка — больший радиус обзора (больше клеток открывается)
   - Логистика — больше очков движения
   - Следопыт — быстрее движение по сложной местности

5. **💰 Проверить экономические навыки**
   - Поместье — ежедневный доход золота
   - Дипломатия — шанс присоединить нейтралов без боя

6. **⬆️ Повышение уровня**
   - Победите сильного врага (много опыта)
   - При повышении уровня — UI выбора навыка
   - 2 навыка на выбор с красивыми карточками

## 🔥 Ключевые технические решения

### Singleton HeroManager
```typescript
const heroManager = HeroManager.getInstance(seed);
```
Гарантирует, что во всех сценах используется один и тот же экземпляр с одними и теми же данными навыков.

### Data-Driven подход
Все 16 навыков описаны в `skills.json` — можно легко добавлять новые без изменения кода.

### Интеграция через геттеры
```typescript
// Вместо прямого изменения кода — используем геттеры
heroManager.getMeleeDamageMultiplier(hero);
heroManager.getVisionRadius(hero);
heroManager.getDailyGoldIncome(hero);
```

### Специализации через switch
Каждый класс героя имеет уникальные бонусы, которые применяются автоматически при создании.
