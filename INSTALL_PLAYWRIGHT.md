# 🔧 Установка Playwright для автотестов

## Проблема
Playwright MCP ожидает версию 1.57.0 (chromium-1200), но установлена 1.60.0 (chromium-1223).

## Решение

### Вариант 1: Автоматический (рекомендуется)
Откройте терминал в папке проекта и выполните:

```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
.\playwright-fix.bat
```

### Вариант 2: Ручной
```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
npm install --save-dev playwright@1.57.0 @playwright/test@1.57.0
npx playwright install chromium
```

### Вариант 3: Без Playwright (тестирование вручную)
Если не хотите устанавливать Playwright, я могу:
- Прочитать весь код статически
- Найти потенциальные баги
- Предоставить чеклист для ручного тестирования

## После установки

Запустите игру:
```bash
npm run dev
```

Затем скажите мне **"готово"** и я проведу полный автотест через Playwright:
- 📸 Скриншоты всех сцен
- 🎮 Тестирование механик
- 🐛 Поиск runtime ошибок
- 📊 Детальный отчёт

## Что будет тестироваться

1. **Главное меню** — запуск, кнопки
2. **Карта (surface)** — генерация, объекты, движение
3. **Подземелье** — переключение (U), порталы
4. **Бой** — начало, магия, мораль/удача
5. **Город** — все 8 вкладок
6. **Save/Load** — F5/F9
7. **Console** — поиск ошибок

## Альтернатива

Если Playwright не работает, выберите:
- **"Статический анализ"** — я прочту код и найду баги
- **"Я протестирую"** — вы опишете проблемы, я исправлю
- **"Продолжай писать"** — пропустить тесты, перейти к следующей задаче
