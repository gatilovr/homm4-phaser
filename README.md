# ⚔️ Heroes IV — Phaser Edition

Браузерная версия Heroes of Might and Magic IV на **Phaser 3.80 + TypeScript + Vite**.

---

## 🚀 Быстрый старт

### 1. Установите зависимости

```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
npm install
```

### 2. Запустите dev-сервер

```bash
npm run dev
```

### 3. Откройте браузер

После запуска в терминале появится строка вроде:

```
  VITE v5.1.0  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

**Откройте URL из терминала!** Обычно это `http://localhost:3000/`.

## 🔧 Устранение проблем

### ❌ "Страница не найдена" / "connectionfailure"

**Причина 1: Сервер не запустился**
Проверьте терминал — там должны быть строки:
```
VITE v5.x.x  ready in XXX ms
➜  Local:   http://localhost:3000/
```
Если их нет — есть ошибка. Скопируйте её и покажите мне.

**Причина 2: Порт 3000 занят**
Vite автоматически переключится на другой порт (3001, 3002...).  
Смотрите на URL в терминале!

**Причина 3: Firewall блокирует**
Разрешите Node.js в брандмауэре Windows:
- Панель управления → Брандмауэр → Разрешение взаимодействия с приложением
- Добавьте `Node.js`

**Причина 4: Ошибка в коде**
Откройте **консоль браузера (F12)** и посмотрите на красные ошибки.

### ❌ Ошибки TypeScript при `npm run build`

Для dev-режима это не критично — Vite/esbuild игнорирует ошибки типов.

### ❌ Белый экран

1. Откройте консоль (F12)
2. Найдите строку `🎮 Starting Heroes IV - Phaser Edition...`
3. Если её нет — ошибка в `main.ts`
4. Если есть `✅ Game instance created successfully!` — ошибка в сценах

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| **ЛКМ на тайл** | Движение героя (A* pathfinding) |
| **ПКМ** | Информация о тайле |
| **Колесо мыши** | Зум камеры (0.5x — 2.5x) |
| **Enter** | Конец хода |
| **H** | Информация о герое |
| **Esc** | Вернуться в меню |
| **Клик на мини-карту** | Перемещение камеры |

### В бою

| Клавиша | Действие |
|---------|----------|
| **ЛКМ на зелёную клетку** | Перемещение юнита |
| **ЛКМ на красного врага** | Атака |
| **Space** | Пропустить ход |
| **D** | Защита |
| **Esc** | Сбежать |

## 📂 Структура проекта

```
hommm4-phaser/
├── index.html              ← Точка входа
├── package.json            ← Зависимости
├── vite.config.ts          ← Настройки Vite
├── tsconfig.json           ← Настройки TypeScript
│
├── src/
│   ├── main.ts             ← Инициализация Phaser
│   ├── config.ts           ← Константы игры
│   ├── types.ts            ← TypeScript типы
│   │
│   ├── scenes/
│   │   ├── BootScene.ts    ← Генерация текстур
│   │   ├── PreloadScene.ts ← Загрузка JSON данных
│   │   ├── MenuScene.ts    ← Главное меню
│   │   ├── WorldScene.ts   ← Карта приключений 60×60
│   │   ├── BattleScene.ts  ← Тактический бой 15×11
│   │   ├── TownScene.ts    ← Управление городом
│   │   └── UIScene.ts      ← Мини-карта, управление
│   │
│   └── utils/
│       ├── Pathfinder.ts   ← A* с 8 направлениями
│       ├── NoiseGenerator.ts ← Perlin noise
│       ├── Random.ts       ← Seed-based генератор
│       └── EventBus.ts     ← Система событий
│
└── public/assets/data/
    ├── factions.json       ← 6 фракций
    ├── spells.json         ← 19 заклинаний
    ├── artifacts.json      ← 10 артефактов
    └── buildings.json      ← 14 зданий
```
