# 💾 Система Save/Load — ПОЛНОСТЬЮ РЕАЛИЗОВАНА

## 🎉 Версия 2.4.0

Сохранение игры — критически важная фича. Теперь можно:
- ✅ **Сохранять** игру в 3 слота
- ✅ **Загружать** из любого слота
- ✅ **Автосохранение** каждый ход
- ✅ **Быстрое сохранение/загрузка** (F5/F9)
- ✅ **Экспорт/импорт** сохранений

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|

## 📦 Что сохраняется

### 🎯 Основные данные
- **Версия сохранения** (для миграций при обновлениях)
- **Timestamp** — дата и время сохранения
- **Seed карты** — для восстановления карты
- **День и неделя** — время в игре
- **Ресурсы** — все 7 типов (gold, wood, ore, crystals, gems, sulfur, mercury)

### 🦸 Герои (все герои игрока)
- Имя, класс, фракция, уровень, опыт
- Позиция (x, y)
- Очки движения (текущие и максимальные)
- **Характеристики**: АТК, ЗАЩ, SP, Знания
- **Навыки** с уровнями и эффектами (все 16 навыков)
- **Экипировка** (10 слотов)
- **Армия** (все существа с количеством)
- Мана (текущая и максимальная)
- Заклинания
- Мораль, удача
- Специализация

### 🏰 Города
- ID, позиция, имя, фракция
- Владелец (игрок / ИИ)
- **Построенные здания** (полный список)
- **Гарнизон** (армия в городе)
- Ресурсы города
- Уровень магической гильдии
- Предложения гильдии
- Существ для найма (с количеством)

### ⛏️ Шахты
- ID, позиция
- **Тип** (золото, дерево, руда, кристаллы, самоцветы, сера, ртуть)
- Владелец

### 🗺️ Объекты карты
- Все 70+ объектов (сундуки, святилища, алтари, школы магии и т.д.)
- Состояние: собран/посещён
- Данные объекта

### 🤖 ИИ противники
- ID, имя, фракция
- Города (ID)
- Герои (с полными данными)
- Шахты (ID)
- Ресурсы ИИ

### 🌫️ Туман войны
- Матрица `visible` — что видит герой сейчас
- Матрица `visited` — что было исследовано

### 🚚 Караваны
- Состояние системы караванов

## 🔧 Технические детали

### Хранение
- **localStorage** (5-10 MB в зависимости от браузера)
- **Ключи**: 
  - `hommm4_save_slot_1`
  - `hommm4_save_slot_2`
  - `hommm4_save_slot_3`
  - `hommm4_autosave`

### Размер сохранения
- Типичный размер: **30-80 KB** (JSON)
- Карта 60×60 с 70+ объектами и 3 героями
- Сжимается до ~10 KB в base64 для экспорта

### Версионирование
Каждое сохранение содержит `version` (сейчас `2.4.0`).  
При загрузке система проверяет версию и применяет миграции если нужно:

```typescript
private migrateIfNeeded(data: SaveData): SaveData {
  if (data.version === SAVE_VERSION) return data;
  
  // Миграции для старых версий
  // if (data.version < '3.0.0') { ... }
  
  data.version = SAVE_VERSION;
  return data;
}
```

### Процесс сохранения

```
1. Пользователь нажимает F5 / Ctrl+S
            ↓
2. SaveSystem.collectSaveData(worldScene)
   Собирает: героев, города, шахты, объекты, ИИ, туман
            ↓
3. JSON.stringify(data)
            ↓
4. localStorage.setItem(key, json)
            ↓
5. Показ уведомления "💾 Игра сохранена"
```

### Процесс загрузки

```
1. Пользователь нажимает F9 / Ctrl+L
            ↓
2. Выбор слота в UI
            ↓
3. SaveSystem.loadGame(slot)
            ↓
4. JSON.parse(json)
            ↓
5. migrateIfNeeded(data) — миграция если старая версия
            ↓
6. scene.start('WorldScene', { loadSave: true, saveData, seed })
            ↓
7. WorldScene.create()
   - Генерирует карту по seed (та же самая!)
   - Создаёт объекты
   - Вызывает saveSystem.restoreGameState(this, data)
            ↓
8. restoreGameState:
   - Восстанавливает день/неделю
   - Восстанавливает ресурсы
   - Восстанавливает героев (позиции, статы, навыки)
   - Восстанавливает города (постройки, гарнизон)
   - Восстанавливает шахты (владельцы)
   - Восстанавливает объекты (посещённые)
   - Восстанавливает ИИ
   - Восстанавливает туман войны
            ↓
9. updateUI() — обновить визуал
            ↓
10. Показ уведомления "✅ Игра загружена"
```

## 🔄 Автосохранение

Автосохранение происходит:
- ✅ **После каждого хода** (кнопка "Конец хода" или Enter)
- ✅ **Через 1 секунду после начала новой игры** (первичный сейв)

Автосохранение хранится в отдельном ключе `hommm4_autosave` и не перезаписывает слоты игрока.

---

## 🛡️ Обработка ошибок

1. **localStorage недоступен** → `isStorageAvailable()` вернёт `false`
2. **Слот пуст** → `loadGame()` вернёт `null`, UI покажет "Слот пуст"
3. **Повреждённый JSON** → try/catch вернёт `null` и покажет "❌ Ошибка загрузки"
4. **Старая версия** → автоматически применит миграции

## 🎯 Что тестировать

1. **Быстрое сохранение (F5)**
   - Нажмите F5 → увидите уведомление "💾 Быстрое сохранение (слот 1)"
   - Обновите страницу (F5 в браузере)
   - Нажмите F9 → игра загрузится с того же места

2. **Диалог сохранения (Ctrl+S)**
   - Нажмите Ctrl+S → откроется окно с 3 слотами
   - Кликните на пустой слот → игра сохранится
   - Информация о слоте отобразится (герой, уровень, день)

3. **Диалог загрузки (Ctrl+L)**
   - Нажмите Ctrl+L → окно со слотами
   - Кликните на слот с сохранением → игра загрузится
   - Клик на пустой слот → уведомление "Слот пуст"

4. **Удаление сохранения**
   - В диалоге нажмите 🗑️ на слоте
   - Подтвердите → слот очистится

5. **Автосохранение**
   - Нажмите "Конец хода" несколько раз
   - В меню → "Загрузить" → увидите "⚡ Автосохранение"
   - Загрузите — игра продолжится с последнего хода

6. **Загрузка из главного меню**
   - Выйдите в меню (через экран победы или перезапуск)
   - Нажмите "💾 ЗАГРУЗИТЬ"
   - Выберите слот → игра продолжится

## 🐛 Известные ограничения

- ⚠️ Сохранения хранятся в **localStorage браузера** — не синхронизируются между устройствами
- ⚠️ Очистка кэша/cookies **удалит все сохранения**
- ⚠️ Приватный режим браузера **не сохраняет** данные между сессиями
- ⚠️ Максимальный размер localStorage ~5-10 MB
