## 🎮 Как играть

### Запуск:
```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
npm install
npm run dev
```

### Что делать:
1. 🗺️ **Исследуйте карту** — кликайте по тайлам
2. 🏰 **Зайдите в город** — стройте, нанимайте, торгуйте
3. ⚔️ **Сражайтесь с нейтралами** — получайте опыт
4. 🤖 **Следите за ИИ** — он развивается каждый ход!
5. 🚨 **Защищайте города** — ИИ будет их атаковать
6. 🏆 **Захватывайте города ИИ** — условие победы!

### Горячие клавиши:
| Клавиша | Действие |
|---------|----------|
| **ЛКМ** | Движение / Взаимодействие |
| **ПКМ** | Инфо о тайле |
| **Enter** | Конец хода |
| **H** | Инфо о герое |
| **Esc** | Меню |
