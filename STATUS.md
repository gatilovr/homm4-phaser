## 📂 Структура проекта

```
hommm4-phaser/
├── src/
│   ├── main.ts              # Точка входа
│   ├── config.ts            # Конфигурация
│   ├── types.ts             # TypeScript типы (320 строк)
│   │
│   ├── scenes/              # 7 игровых сцен
│   │   ├── BootScene.ts     # Загрузка и генерация текстур
│   │   ├── PreloadScene.ts  # Загрузка ресурсов
│   │   ├── MenuScene.ts     # Главное меню
│   │   ├── WorldScene.ts    # Карта приключений
│   │   ├── BattleScene.ts   # ⚔️ Бой (1500 строк, 100%)
│   │   ├── TownScene.ts     # Управление городом
│   │   └── UIScene.ts       # Мини-карта
│   │
│   ├── systems/             # 9 игровых систем
│   │   ├── SpellSystem.ts         # 19 заклинаний
│   │   ├── BattleEffects.ts       # Анимации и эффекты
│   │   ├── MoraleLuck.ts          # Мораль и удача
│   │   ├── BattleAI.ts            # ИИ в бою
│   │   ├── CreatureTypes.ts       # Типы юнитов
│   │   ├── CreatureAbilities.ts   # 30+ способностей ✨
│   │   ├── Necromancy.ts          # Поднятие нежити ✨
│   │   ├── SiegeSystem.ts         # Осада городов ✨
│   │   └── BattleQueue.ts         # Очередь ходов ✨
│   │
│   └── utils/               # Утилиты
│       ├── Pathfinder.ts    # A* pathfinding
│       ├── NoiseGenerator.ts # Perlin noise
│       ├── Random.ts        # Seed-based RNG
│       └── EventBus.ts      # События
│
└── public/assets/data/      # JSON данные
    ├── factions.json        # 6 фракций
    ├── spells.json          # 19 заклинаний
    ├── artifacts.json       # 10 артефактов
    └── buildings.json       # 14 зданий
```

## 🎮 Что можно делать в игре

### Карта:
- Клик по тайлу — герой идёт по A* пути
- Колесо мыши — зум (0.5x — 2.5x)
- Подбор ресурсов, артефактов
- Захват шахт
- Вход в города

### Бой (100% готов!):
- Клик по зелёным клеткам — движение
- Клик по красным — атака
- M — книга магии (19 заклинаний)
- W — ждать, D — защита
- A — авто-бой
- S — сдаться, Esc — сбежать
- Все 30+ способностей работают
- Некромантия поднимает нежить
- Осада городов со стенами

### Город:
- Просмотр зданий
- Строительство
