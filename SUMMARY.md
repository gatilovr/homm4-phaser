## 🎮 Как играть

### Исследование карты
1. **Кликните на тайл** — герой пойдёт по A* пути
2. **Исследуйте туман войны** — открывайте новые территории
3. **Собирайте ресурсы** — кликните на кучку ресурсов
4. **Захватывайте шахты** — подходят к шахте, получаете доход

### Взаимодействие с объектами
1. **Подойдите к объекту** — герой автоматически взаимодействует
2. **Получите награду** — уведомление с описанием эффекта
3. **Объект исчезает** — большинство объектов одноразовые
4. **Постоянные объекты** — мельницы и колёса дают еженедельный доход

### Стратегия
- **Шахты** — захватывайте рано для стабильного дохода
- **Школы магии** — ищите для расширения книги заклинаний
- **Святилища** — посещайте для усиления героя
- **Сундуки** — выбирайте золото если нужна экономика, опыт если нужна прокачка
- **Лагеря беженцев** — бесплатная армия, захватывайте рано
- **Мельницы/колёса** — долгосрочный доход, захватывайте если планируете долгую игру
