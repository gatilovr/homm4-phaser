## 🚀 Как запустить

### Шаг 1: Откройте терминал в папке проекта

В VS Code:
1. Откройте папку `C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser`
2. Нажмите `Ctrl+Shift+~` для открытия терминала
3. Или: `File → Open Folder...` → выберите папку

В Windows Terminal/PowerShell:
```bash
cd C:\Users\gatil\VisualStudioCodeProject\hommm4-phaser
```

### Шаг 2: Установите зависимости

```bash
npm install
```

Это скачает:
- Phaser 3.80 (~4MB)
- TypeScript 5.3
- Vite 5.1
- И другие dev-зависимости

**Время:** ~30-60 секунд

### Шаг 3: Запустите dev-сервер

```bash
npm run dev
```

Будет выведено:
```
  VITE v5.1.0  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

### Шаг 4: Откройте в браузере

Автоматически откроется `http://localhost:3000`

Или откройте вручную:
- Chrome: `http://localhost:3000`
- Firefox: `http://localhost:3000`
- Edge: `http://localhost:3000`

## 🎮 Что работает прямо сейчас

### ✅ Главное меню
- Красивый интерфейс с анимированными звёздами
- 4 кнопки: Новая игра, Загрузить, Настройки, Об игре
- Hover-эффекты и анимации

### ✅ Карта приключений (WorldScene)
- **Процедурная генерация 60×60** — Perlin noise
- **8 типов тайлов** — трава, песок, вода, скалы, снег, болото, лава, лес
- **A* pathfinding** — кликните на тайл → герой пойдёт по оптимальному пути
- **Камера следит за героем** — плавное следование
- **Зум колесом мыши** — от 0.5x до 2.5x
- **Туман войны** — территория раскрывается при исследовании
- **Объекты** — города, шахты, артефакты, существа, ресурсы, порталы
- **Взаимодействие** — клик на объект → соответствующее действие

### ⚔️ Бой (BattleScene)
- **Поле 15×11** — тактическая сетка
- **Система ходов по скорости** — быстрые юниты ходят первыми
- **Перемещение** — зелёная подсветка доступных клеток (BFS)
- **Атака** — клик на врага рядом → автоматическая атака
- **Контратаки** — защитник отвечает на удар
- **Анимации** — tween для перемещения и ударов
- **ИИ противника** — атакует ближайших врагов
- **Герои как юниты** — ключевая фича HoMM4!
- **UI боя** — лог, информация о юнитах, кнопки управления

### 🏰 Город (TownScene)
- **Визуальный интерфейс** — замок с башнями
- **14 зданий** — Цитадель, Казармы, Стрельбище, Рынок, Кузница...
- **Строительство** — клик на здание → построить (если хватает ресурсов)
- **Требования** — каскадные (нужна Цитадель для Казарм)
- **Отображение ресурсов** — золото, дерево, руда

### 🗺️ Мини-карта (UIScene)
- **Реальное время** — обновляется каждый кадр
- **Кликабельная** — кликните для перемещения камеры
- **Показывает** — тайлы, объекты, позицию героя

## 🎯 Управление

### Карта
- **ЛКМ** — движение героя (A* pathfinding)
- **ПКМ** — информация о тайле
- **Колесо мыши** — зум (0.5x — 2.5x)
- **Enter** — конец хода
- **H** — информация о герое
- **Esc** — вернуться в меню

### Бой
- **ЛКМ на клетку** — перемещение
- **ЛКМ на врага** — атака (если рядом)
- **Space** — пропустить ход
- **D** — защита
- **Esc** — сбежать

### Город
- **ЛКМ на здание** — построить
- **Esc** — вернуться на карту

## 🔧 Технические детали

### Стек технологий
- **Phaser 3.80** — игровой движок (WebGL)
- **TypeScript 5.3** — строгая типизация
- **Vite 5.1** — быстрая сборка
- **Node.js 18+** — окружение

### Архитектура
- **Scene-based** — 7 сцен (Boot, Preload, Menu, World, Battle, Town, UI)
- **Event-driven** — EventBus для коммуникации
- **Data-driven** — весь контент в JSON файлах
- **Type-safe** — TypeScript strict mode

### Производительность
- **WebGL** — аппаратное ускорение
- **60 FPS** — плавный рендеринг
- **Lazy loading** — ресурсы загружаются по требованию
