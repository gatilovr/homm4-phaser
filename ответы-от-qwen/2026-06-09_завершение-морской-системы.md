## 🎮 Как работает верфь

### 📋 Требования для покупки корабля:
1. ✅ Построено здание `shipyard` (💰2000 + 🪵20 дерева)
2. ✅ Есть **вода** в радиусе 2 клеток от города (проверка `canBuildShipyard`)

### 💰 Покупка корабля:
- Стоимость: **1000 золота** (канон HoMM4)
- Корабль размещается на **ближайшей свободной водной клетке** рядом с городом
- Поиск в расширяющемся радиусе (1-5 клеток от города)
- Если вода не найдена — золото возвращается

### 📊 UI вкладки "⚓ Верфь":
- Заголовок с описанием
- Большая иконка 🚢 (лодка)
- Название "Лодка (Boat)"
- Описание функций
- Статы (вместимость 7 слотов, базовая скорость)
- Цена и текущее золото
- Кнопка "🚢 КУПИТЬ" / "❌ Нет золота"
- Инструкция по использованию (посадка/высадка/водовороты)
- Статистика: сколько героев на кораблях

### ⚠️ Защитные проверки:
- Если верфь не построена → "⚓ Постройте Верфь для покупки кораблей"
- Если нет воды рядом → "🌊 Верфь не может строить — нет воды рядом с городом!"
- Если нет золота → кнопка неактивна + уведомление

## 🎮 Как протестировать

### Сценарий 1: Купить корабль в верфи
1. Зайти в свой город
2. Вкладка "🏗️ Здания" → построить Верфь (💰2000 + 🪵20)
3. Вкладка **"⚓ Верфь"** (клавиша 9)
4. Кликнуть "🚢 КУПИТЬ" (💰1000)
5. Уведомление: "✅ Корабль построен на клетке (X, Y)!"
6. Выйти из города (ESC)
7. Найти корабль на воде рядом с городом
8. Подойти к нему и кликнуть — **герой сядет на борт!**

### Сценарий 2: Движение по воде
- На корабле: клик по водной клетке → герой движется
- Клик по берегу → **высадка** (корабль остаётся на воде)
- Клик по whirlpool → **телепортация** (25% потеря отряда!)

### Сценарий 3: Морские объекты
- Корабли (boat) → посадка
- Водовороты (whirlpool) → телепортация
- Затонувшие корабли (shipwreck) → сокровища / бой
- Морские сундуки (sea_chest) → 1000-2500 золота
- Мусор (flotsam) → дерево или золото
- Бутылки (bottle) → послания
