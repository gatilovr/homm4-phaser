## 🎮 Как использовать

### Открытие книги заклинаний:
- **Клавиша M** на карте
- Или клик по кнопке 📖 (если добавлена в UI)

### Применение заклинаний:
1. Откройте книгу (M)
2. Кликните на карточку заклинания
3. Для Dimension Door — выберите подсвеченную клетку
4. Для Town Portal — герой телепортируется автоматически
5. Для Fly/Water Walk — эффект действует до конца дня

### Горячие клавиши:
| Клавиша | Действие |
|---------|----------|
| **M** | Открыть/закрыть книгу заклинаний |
| **ESC** | Отменить выбор цели (Dimension Door) |
| **Enter** | Конец дня (сбрасывает Fly/Water Walk) |

## 🔧 Технические детали

### Архитектура:
```
AdventureMagicSystem.ts (логика)
    ↓
AdventureMagicBook.ts (UI)
    ↓
WorldScene.ts (интеграция)
```

### Ключевые методы:

**AdventureMagicSystem:**
- `canCastSpell()` — проверка возможности применения
- `castTownPortal()` — телепорт в город
- `castDimensionDoor()` — телепорт на клетку
- `castFly()` / `castWaterWalk()` — дневные эффекты
- `castSummonBoat()` / `castScuttleBoat()` — работа с кораблями
- `castVisions()` / `castIdentifyHero()` — разведка
- `resetDailyEffects()` — сброс в конце дня

**WorldScene:**
- `initAdventureMagic()` — инициализация
- `onSpellSelected()` — обработчик выбора
- `castTownPortal()` / `castDimensionDoor()` — применение
- `createMagicEffect()` / `createTeleportEffect()` — визуал
- `resetDailyMagicEffects()` — сброс при конце дня

**AdventureMagicBook:**
- `show()` / `hide()` / `toggle()` — управление видимостью
- `createSpellCards()` — создание карточек 4×2
- `showTooltip()` — описание заклинания при hover

## 🎨 Визуальные эффекты

### Магическая вспышка (createMagicEffect):
- Расширяющийся фиолетовый круг
- Золотое ядро
- Длительность: 500ms

### Телепортация (createTeleportEffect):
- 8 вращающихся голубых частиц
- Центральная белая вспышка
- Fade-out героя при телепортации

### Подсветка целей (Dimension Door):
- Зелёные полупрозрачные квадраты
- Контур 2px
- Только проходимые клетки без объектов

## 📚 Документация

### Канон HoMM4:
В оригинальной игре Heroes of Might and Magic IV (2002) магия на карте включала:
- Town Portal (телепорт в город)
- Dimension Door (телепорт на N клеток)
- Fly (полёт)
- Water Walk (хождение по воде)
- Summon Boat (призыв корабля)
- Scuttle Boat (потопление корабля)
- Visions (разведка)
- Identify Hero (опознание героя)

Все 8 заклинаний реализованы в соответствии с каноном.
