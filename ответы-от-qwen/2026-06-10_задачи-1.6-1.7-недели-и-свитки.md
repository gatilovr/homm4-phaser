```typescript
if (this.weeksSystem) {
  const weekMultiplier = this.weeksSystem.getCreatureGrowthMultiplier(creatureId);
  growth = Math.floor(growth * weekMultiplier);

```

**d) В методе `endTurn()` — бонус дохода золота:**
```typescript
// НОВОЕ: применяем модификатор от недели gold_abundance
if (this.weeksSystem && townIncomeTotal > 0) {
  const goldBonusPercent = this.weeksSystem.getGoldIncomeBonusPercent();
  if (goldBonusPercent > 0) {
    const bonus = Math.floor(townIncomeTotal * goldBonusPercent / 100);
    townIncomeTotal += bonus;
  }
}
```

### 🎮 Как работает в игре:

1. **Каждые 7 дней** (при нажатии Enter) вызывается `endTurn()`
2. `weeksSystem.nextDay()` переключает день/неделю
3. Если сменилась неделя — генерируется новая случайная неделя
4. Показывается уведомление: `📅 Новая неделя! [Название] [Описание]`
5. Применяется `applyWeeklyGrowth()` с модификатором от недели
6. Доход золота увеличивается на процент от недели `gold_abundance`

## 📜 Задача 1.7: Магические свитки (Magic Scrolls)

### ✅ Выполнено:

#### 1. Файл `MagicScrollSystem.ts` создан ✅ СДЕЛАНО СЕЙЧАС

Полная система магических свитков (350+ строк):

**Основные функции:**
- `generateRandomScroll(minLevel, maxLevel)` — генерация случайного свитка
- `generateMultipleScrolls(count, minLevel, maxLevel)` — генерация нескольких
- `canUseScroll(hero, scroll)` — проверка возможности использования
- `useScroll(hero, scroll, targetId)` — использование свитка в бою
- `addScrollToHero(hero, scroll)` — добавление свитка герою
- `removeScrollFromHero(hero, scrollId)` — удаление после использования
- `getScrollValue(scroll)` — расчёт стоимости в золоте
- `getRarityColor(rarity)` — цвет для UI

**25 свитков в 5 школах магии:**

**Life (Жизнь):**
- Исцеление, Благословение, Лечение, Воскрешение, Ангел-хранитель

**Death (Смерть):**
- Проклятие, Слабость, Волна смерти, Имплозия, Оживление мёртвых

**Order (Порядок):**
- Ускорение, Замедление, Щит, Телепорт, Клонирование

**Chaos (Хаос):**
- Огненный шар, Молния, Цепная молния, Метеорит, Армагеддон

**Natural (Природа):**
- Каменная кожа, Жажда крови, Призыв волков, Призыв элементалей, Лианы

**4 уровня редкости:**
- Common (обычный) — серый, вес 15
- Uncommon (необычный) — зелёный, вес 10-12
- Rare (редкий) — синий, вес 3-5
- Epic (эпический) — фиолетовый, вес 1-2
